Vueでリアクティブなデータバインディング

📅 2025年10月4日 👤 vue_master ⏱️ 7分で読める

Vue 3のComposition APIは、ロジックの再利用性を大幅に向上させます。 refとreactiveを使ったリアクティブなデータ管理の基本を学びましょう。

Composition APIの基礎

Vue 3 
<script setup>
import { ref, reactive, computed } from 'vue'

// refを使ったプリミティブ値の管理
const count = ref(0)
const message = ref('Hello Vue!')

// reactiveを使ったオブジェクトの管理
const user = reactive({
  name: '山田太郎',
  age: 25,
  email: 'yamada@example.com'
})

// 算出プロパティ
const doubleCount = computed(() => count.value * 2)

// メソッド
const increment = () => {
  count.value++
}

const updateUser = () => {
  user.age++
}
</script>

テンプレートでの使用

Vue Template 
<template>
  <div class="container">
    <h2>{{ message }}</h2>
    
    <div class="counter">
      <p>カウント: {{ count }}</p>
      <p>2倍: {{ doubleCount }}</p>
      <button @click="increment">
        増やす
      </button>
    </div>

    <div class="user-info">
      <p>名前: {{ user.name }}</p>
      <p>年齢: {{ user.age }}</p>
      <button @click="updateUser">
        年齢を増やす
      </button>
    </div>
  </div>
</template>
