Scalaで関数型プログラミング

Scalaは、オブジェクト指向と関数型プログラミングを融合した強力な言語です。 パターンマッチング、高階関数、不変データ構造を活用して、堅牢なコードを書きましょう。

ケースクラスとパターンマッチング

Scala 
sealed trait Result[+A]
case class Success[A](value: A) extends Result[A]
case class Failure(error: String) extends Result[Nothing]

def divide(a: Int, b: Int): Result[Double] = {
  if (b == 0)
    Failure("ゼロ除算エラー")
  else
    Success(a.toDouble / b)
}

// パターンマッチングの使用
val result = divide(10, 2) match {
  case Success(value) => 
    println(s"結果: $value")
  case Failure(error) => 
    println(s"エラー: $error")
}

// リストのパターンマッチング
def sum(list: List[Int]): Int = list match {
  case Nil => 0
  case head :: tail => head + sum(tail)
}

高階関数とコレクション操作

Scala 
case class Person(name: String, age: Int)

val people = List(
  Person("山田", 25),
  Person("佐藤", 30),
  Person("鈴木", 22)
)

// map, filter, reduceの組み合わせ
val averageAge = people
  .filter(_.age > 20)
  .map(_.age)
  .reduce(_ + _) / people.length.toDouble

// for式による表現
val adultNames = for {
  person <- people
  if person.age >= 20
} yield person.name

// カリー化された関数
def add(x: Int)(y: Int): Int = x + y

val add5 = add(5) _
val result = add5(3)  // 8

// 高階関数の定義
def applyTwice[A](f: A => A, x: A): A = f(f(x))

val double = (x: Int) => x * 2
applyTwice(double, 3)  // 12
