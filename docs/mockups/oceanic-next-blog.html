Haskellで純粋関数型プログラミング

📅 2025年10月4日 👤 haskell_pure ⏱️ 16分で読める

Haskellは純粋関数型言語の代表格で、遅延評価と強力な型システムが特徴です。 モナドと型クラスを理解することで、抽象的で美しいコードを書けるようになります。

基本的な型と関数

Haskell 
-- 型シグネチャと関数定義
factorial :: Integer -> Integer
factorial 0 = 1
factorial n = n * factorial (n - 1)

-- リスト操作
quicksort :: Ord a => [a] -> [a]
quicksort [] = []
quicksort (x:xs) =
  let smaller = quicksort [a | a <- xs, a <= x]
      bigger  = quicksort [a | a <- xs, a > x]
  in  smaller ++ [x] ++ bigger

-- 高階関数
applyTwice :: (a -> a) -> a -> a
applyTwice f x = f (f x)

-- カリー化された関数
zipWith' :: (a -> b -> c) -> [a] -> [b] -> [c]
zipWith' _ [] _ = []
zipWith' _ _ [] = []
zipWith' f (x:xs) (y:ys) = f x y : zipWith' f xs ys

モナドとdo記法

Haskell 
-- Maybe モナド
safeDivide :: Double -> Double -> Maybe Double
safeDivide _ 0 = Nothing
safeDivide x y = Just (x / y)

calculateAverage :: [Double] -> Maybe Double
calculateAverage [] = Nothing
calculateAverage xs = Just (sum xs / fromIntegral (length xs))

-- IO モナドとdo記法
main :: IO ()
main = do
  putStrLn "名前を入力してください:"
  name <- getLine
  putStrLn ("こんにちは、" ++ name ++ "さん!")

-- Either モナド
data AppError = NetworkError String
              | ParseError String
              deriving Show

processData :: String -> Either AppError Int
processData input = do
  trimmed <- validateInput input
  parsed <- parseNumber trimmed
  return (parsed * 2)
