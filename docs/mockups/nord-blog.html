Rustで高速なCLIツールを作る

📅 2025年10月4日 👤 rust_lover ⏱️ 12分で読める

Rustは安全性とパフォーマンスを両立した言語として注目されています。 今回はclapクレートを使って、本格的なコマンドラインツールを作成します。

プロジェクトのセットアップ

Rust 
use clap::{Parser, Subcommand};
use std::path::PathBuf;

/// シンプルなファイル管理ツール
#[derive(Parser)]
#[command(author, version, about, long_about = None)]
struct Cli {
    /// 操作対象のファイルパス
    #[arg(short, long, value_name = "FILE")]
    file: Option<PathBuf>,

    /// 詳細出力を有効化
    #[arg(short, long)]
    verbose: bool,

    #[command(subcommand)]
    command: Option<Commands>,
}

#[derive(Subcommand)]
enum Commands {
    /// ファイルを圧縮
    Compress {
        #[arg(short, long)]
        output: PathBuf,
    },
    /// ファイルを解凍
    Decompress {
        #[arg(short, long)]
        output: PathBuf,
    },
}

メイン処理の実装

Rust 
fn main() -> Result<(), Box<dyn std::error::Error>> {
    let cli = Cli::parse();

    if cli.verbose {
        println!("詳細モードが有効です");
    }

    match &cli.command {
        Some(Commands::Compress { output }) => {
            println!("圧縮中: {:?} -> {:?}", cli.file, output);
            // 圧縮処理を実装
        }
        Some(Commands::Decompress { output }) => {
            println!("解凍中: {:?} -> {:?}", cli.file, output);
            // 解凍処理を実装
        }
        None => {
            println!("コマンドが指定されていません");
        }
    }

    Ok(())
}
