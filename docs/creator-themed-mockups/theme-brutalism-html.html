[ MANIFESTO ]

RAW DESIGN PHILOSOPHY

ブルータリズムは、装飾を拒否します。生のHTML、最小限のCSS、そして純粋な機能性。ウェブデザインの原点回帰です。美しさの定義を問い直し、洗練を拒否し、本質だけを残す。これがブルータリズムの宣言です。

建築におけるブルータリズム運動（1950-70年代）から着想を得て、ウェブデザインにおけるブルータリズムは、過度に洗練されたデザインへの反抗として2014年頃に登場しました。Le Corbusier の「béton brut（生のコンクリート）」という概念が、デジタル空間で再解釈されたのです。

グラデーションは虚飾です。影は不要です。アニメーションは気晴らしに過ぎません。必要なのは、情報、構造、そして誠実さだけです。ブルータリズムは、デザインを本来あるべき姿に戻します。

[ AESTHETIC ]

ANTI-AESTHETIC MOVEMENT

美しさの定義を問い直す。装飾は不要、余白は贅沢、グラデーションは虚飾。ブルータリストデザインは、従来の美の概念に挑戦します。醜さの中に真実があり、粗さの中に誠実さがあります。

Craigslist、Wikipedia、そして Hacker News。これらのウェブサイトは、視覚的な魅力を犠牲にして機能性を最大化しました。結果として、何百万人ものユーザーに愛される存在となりました。デザインは目的ではなく、手段に過ぎないのです。

ブルータリズムは、デザイナーのエゴを排除します。ユーザーが求めているのは、デザイナーの自己表現ではなく、必要な情報への迅速なアクセスです。過剰なデザインは、しばしばユーザビリティを損ないます。

[ TYPOGRAPHY ]

MONOSPACE SUPREMACY

Courier New、Monaco、Consolas。モノスペースフォントは、ブルータリズムの標準です。すべての文字が同じ幅を持つ。この平等性が、タイポグラフィの民主化を実現します。プログラマーが使うフォントは、最も誠実なフォントです。

可変幅フォントの優雅さを拒否し、固定幅フォントの機能性を選ぶ。これは aesthetic な選択ではなく、哲学的な選択です。すべての文字は平等に扱われるべきであり、視覚的なヒエラルキーは最小限であるべきです。

タイポグラフィにおける装飾、リガチャ、カーニング。これらすべてを拒否します。文字は情報を伝えるためのツールであり、芸術作品ではありません。読みやすさは、様式化された美しさよりも重要です。

[ STRUCTURE ]

HTML IS ENOUGH

HTMLだけで十分です。セマンティックなマークアップ、適切な見出し構造、そしてアクセシブルなコンテンツ。これ以上何が必要でしょうか。CSSフレームワークは肥大化をもたらし、JavaScriptライブラリはパフォーマンスを低下させます。

Tim Berners-Lee が最初に作ったウェブは、シンプルでした。ハイパーリンクで繋がったテキスト文書。それだけで、情報革命が始まりました。私たちは、その本質を見失っていないでしょうか。

ブルータリストウェブサイトは、数キロバイトで完結します。メガバイト単位のJavaScriptバンドルは不要です。速度、アクセシビリティ、そして持続可能性。これらすべてが、シンプルさから生まれます。

Posted: 2025.10.01 | Category: Web Development
[ CONTRAST ]

BLACK AND WHITE

黒と白。これ以上のコントラストはありません。グレースケールは妥協です。中間色は曖昧さです。ブルータリズムは、明確さを求めます。オンかオフか。1か0か。二進法の美学です。

色彩理論は複雑すぎます。色相、彩度、明度。これらすべてを放棄し、最も基本的な対比に戻ります。読みやすさは最大化され、視覚的な気晴らしは最小化されます。情報が前面に出ます。

[ MANIFESTO ]

FUNCTION OVER FORM

形態は機能に従う。Louis Sullivan のこの原則を、ブルータリズムは極限まで推し進めます。機能が唯一の形態です。装飾的な要素はすべて、機能性を損なう可能性があります。

ボタンはボタンらしく見えるべきです。リンクは下線が引かれているべきです。フォームは明確であるべきです。ユーザーを混乱させるクリエイティブなUIは、デザインの失敗です。慣習に従うことは、創造性の欠如ではなく、ユーザーへの敬意です。

Posted: 2025.09.29 | Category: UX Principles
[ LEGACY ]

BRUTALIST TRADITION

建築におけるブルータリズムは、戦後の理想主義を体現していました。安価で、機能的で、誠実な建物。Le Corbusier の Unité d'Habitation、Paul Rudolph の Yale Art and Architecture Building。これらの建築物は、社会正義と実用性を追求しました。

ウェブにおけるブルータリズムも、同じ精神を共有しています。誰もが使える、速く読み込まれる、アクセシブルなウェブサイト。これらは、民主的なウェブの理想です。美しいデザインは富裕層のためのものではなく、機能的なデザインはすべての人のためのものです。

[ CONCLUSION ]

EMBRACE THE RAW

ブルータリズムは、すべての人のためのデザインではありません。それは statement であり、provocation であり、そして alternative です。主流のデザイントレンドに疑問を投げかけ、デザインの本質について考えさせます。

生のコンクリートが美しいように、生のHTMLも美しい。洗練は必ずしも改善ではありません。時には、荒々しさこそが真実です。ブルータリズムは、この真実を受け入れることを私たちに求めます。
